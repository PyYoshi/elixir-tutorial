Elixirチュートリアル
===================

ちょっとElixirが気になったのでチュートリアルとかやる

# phoenix

Full stack

http://www.phoenixframework.org/

## 準備

```bash
$ cd phoenix/app
$ mix deps.get
```

## サーバ実行

```bash
$ mix phoenix.server
```

or

```bash
$ iex -S mix phoenix.server
```
